# 多后端接入设计方案

## 概述

MeoClaw 需要支持同时接入 OpenClaw 和 Hermes Agent 两个后端，用户可自由选择。

## 方案选择

**方案 A：纯 HTTP 轮询**

- OpenClaw: 保持现有 WebSocket 连接（`session.tool` 事件）
- Hermes: HTTP POST + SSE stream
- BackendManager 统一通过 HTTP 检测存活

**推荐理由**: 最小改动，最大兼容。保持 OpenClaw 现有实现不变，Hermes 原生 HTTP/SSE 天然适配。

## 核心 Trait 定义

```rust
// backend/trait.rs

pub struct ToolEvent {
    pub name: String,
    pub phase: ToolPhase,  // start, progress, result, error
    pub args: Option<serde_json::Value>,
    pub preview: Option<String>,
}

pub enum ToolPhase {
    Started,
    Progress,
    Result,
    Error,
}

pub struct StreamResponse {
    pub text_delta: String,       // 增量文本
    pub tool_event: Option<ToolEvent>,  // 工具事件
    pub is_final: bool,          // 是否结束
}

pub trait BackendClient: Send {
    fn name(&self) -> &str;

    // 健康检测
    fn check_health(&self) -> bool;

    // 发送消息，返回 StreamResponse 流
    fn send_message(&self, text: &str) -> Result<Box<dyn Stream>, Error>;

    // 获取配置信息
    fn endpoint(&self) -> &str;
    fn auth_headers(&self) -> HashMap<String, String>;
}
```

## BackendManager 接口

```rust
// backend/manager.rs

pub enum ActiveBackend {
    OpenClaw(OpenClawClient),
    Hermes(HermesClient),

pub struct BackendConfig {
    pub selected: String,  // "openclaw" | "hermes"
    pub openclaw: OpenClawConfig,
    pub hermes: HermesConfig,

pub struct BackendHealth {
    pub available: bool,
    pub backend: String,
    pub endpoint: String,
    pub error: Option<String>,

impl BackendManager {
    pub fn new() -> Self;
    pub fn switch(&mut self, backend: &str) -> Result<(), Error>;
    pub fn check_health(&self) -> BackendHealth;
    pub async fn send_message(&self, text: &str) -> Result<Box<dyn Stream>, Error>;
    pub fn current(&self) -> Option<&str>;
}
```

## HermesClient 实现

```rust
// backend/hermes_client.rs

impl BackendClient for HermesClient {
    fn name(&self) -> &str { "hermes" }

    fn check_health(&self) -> bool {
        // HTTP GET http://127.0.0.1:8642/health
    }

    fn send_message(&self, text: &str) -> Result<Box<dyn Stream>, Error> {
        // POST /v1/chat/completions
        // Body: {"model": "...", "messages": [{"role": "user", "content": text}], "stream": true}
        // 解析 SSE stream:
        //   - delta.content -> StreamResponse.text_delta
        //   - hermes.tool.progress / __tool_started__ / __tool_completed__ -> StreamResponse.tool_event
    }
}
```

## 依赖说明

### OpenClaw
- 端口: 18789
- 协议: WebSocket
- 健康检测: HTTP GET /
- 消息格式: 私有 JSON，session.tool / agent 事件

### Hermes Agent
- 端口: 8642
- 协议: HTTP + SSE
- 健康检测: HTTP GET /health
- 消息格式: OpenAI Chat Completions
- 工具事件: SSE stream 中的 hermes.tool.progress / __tool_started__ / __tool_completed__
