# 选项菜单像素风格重构

> 更新日期: 2026-04-22
> 状态: 当前实现

## 概述

将选项菜单从「现代圆润风格」更新为「像素复古风格」，参照设计 `.superpowers/brainstorm/35802-1776787960/content/pixel-meo-light-v2.html`。界面全新改版，功能逻辑保持不变。

## 改动内容

### 布局变更

**原布局：** 垂直堆叠卡片，所有面板一次性展示

**新布局：** 侧边栏 + 内容面板
```
┌──────────────────────────────────────┐
│  ● ● ●      MEOCLAW OPTIONS          │ ← 标题栏（红绿灯圆点）
├──────────┬───────────────────────────┤
│          │                           │
│  </>     │     [SIZE/BACKEND/        │
│ MEO-CLAW │      SOUND 面板内容]       │
│          │                           │
│  SIZE    │                           │
│  BACKEND │                           │
│  SOUND   │                           │
│          │                           │
│  v1.0.0  │                           │
└──────────┴───────────────────────────┘
```

### 视觉风格

| 元素 | 原样式 | 新样式 |
|------|--------|--------|
| 字体 | Plus Jakarta Sans | Press Start 2P（标题/nav）+ monospace（正文）|
| 边框 | 2px, border-radius 28px | 3px, border-radius 12px |
| 配色 | 毛玻璃 + 渐变 | 像素暖橙色系 |
| 导航 | 无 | 左侧 sidebar，选中项橙色背景 |

### 配色方案

```
#fff3dc  — 背景浅
#ffd59f  — 背景深
#fd811e  — 主橙色（选中/强调）
#d07b1f  — 边框
#934500  — 深橙文字
#865012  — 次要文字
#fffaf5  — 面板背景
```

### 窗口尺寸调整

| 属性 | 原值 | 新值 |
|------|------|------|
| inner_size | 520x420 | 680x550 |
| min_inner_size | 480x380 | 620x480 |
| CSS max-width | 560px | 720px |
| 内容区高度 | 340px | 440px |

### 功能保留

- ✅ 后端切换逻辑（OpenClaw/Hermes 选择 + 确认对话框）
- ✅ 音效选择逻辑（Off/Chime/Soft/Notification/Classic/Crisp + 预览播放）
- ✅ 缩放滑块（50%-200%）
- ✅ Toast 提示（成功/失败）
- ✅ 确认对话框
- ✅ 所有 localStorage 键名不变
- ✅ 所有 event 名称不变

## 技术细节

### 面板切换

通过 `data-panel` 属性关联 nav 和 content，点击 nav 时切换 `active` class。

### 后端状态显示修复

将状态指示器（灯）和状态文字分开：
```html
<div class="backend-status">
  <div class="selector-indicator running" id="openclaw-indicator"></div>
  <span class="status-text" id="openclaw-text">ON</span>
</div>
```
