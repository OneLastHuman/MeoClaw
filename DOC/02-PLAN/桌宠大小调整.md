# 桌宠大小调整功能

## 概述

通过选项窗口中的滑块控制，实时调整桌宠在桌面上的显示大小和所有内部元素。

## 实现方式

### 缩放策略

窗口物理尺寸和所有内部元素同步缩放，确保：
- 窗口尺寸随缩放系数变化
- 所有元素（猫咪、输入框、气泡等）同步缩放
- 内容不被裁剪

```
┌─────────────────────────────────────────────────┐
│  用户拖动滑块 (50% ~ 200%)                      │
│         ↓                                       │
│  保存到 localStorage                            │
│         ↓                                       │
│  petScale 更新 → CSS 变量更新                   │
│         ↓                                       │
│  窗口尺寸 + 所有元素同步缩放                    │
└─────────────────────────────────────────────────┘
```

### 缩放范围

| 最小值 | 默认值 | 最大值 | 步进 |
|--------|--------|--------|------|
| 50%    | 100%   | 200%   | 10%  |

## 核心设计：CSS 变量方案

### 可扩展的缩放变量（petSettings.ts）

```typescript
export const scaledCSSVars = computed(() => {
  const s = petScale.value
  return {
    // 基础尺寸
    catWidth: 180 * s,
    catHeight: 180 * s,

    // 窗口尺寸
    windowWidth: 180 * s,
    windowHeight: 180 * s,

    // Input Overlay
    inputBottom: 35 * s,
    inputWidth: 160 * s,
    inputHeight: 42 * s,
    inputFontSize: 16 * s,

    // FileDropOverlay
    fileDropTop: 70 * s,
    fileIconSize: 40 * s,

    // Working Bubble
    workingBubbleLeft: 100 * s,
    workingBubbleTop: 30 * s,

    // Queue Bubble
    queueBubbleTop: 150 * s,

    // Response Bubble
    responseBubbleLeft: 120 * s,
    responseBubbleBottom: 150 * s,
    responseBubbleWidth: 360 * s,
    responseBubbleHeight: 500 * s,
    // ... 可继续扩展
  }
})
```

### 注入 CSS 变量（DesktopPet.vue）

```typescript
const rootCSSVars = computed(() => {
  const v = scaledCSSVars.value;
  return {
    '--cat-width': `${v.catWidth}px`,
    '--window-width': `${v.windowWidth}px`,
    '--input-bottom': `${v.inputBottom}px`,
    // ... 所有变量
  };
});
```

### 使用 CSS 变量（模板和样式）

```vue
<div class="pet-root" :style="rootCSSVars">
  <!-- 子元素通过 CSS 变量引用 -->
  <div class="input-overlay">...</div>
</div>
```

```css
.input-overlay {
  bottom: var(--input-bottom);
  width: var(--input-width);
}
```

## 可扩展性

新增状态/元素时，只需在 `scaledCSSVars` 中添加新的变量，然后在对应元素的 CSS 中使用即可。

## 窗口间同步

当选项窗口修改 localStorage 时，主窗口通过以下方式同步：

1. `storage` 事件监听（跨标签页同步）
2. `pet-scale-changed` Tauri 事件（选项窗口主动通知）

## 视觉样式

选项窗口采用与 MeoClaw Arcade 一致的橙色主题：

- 主色调：`#fd811e` (橙)
- 强调色：`#934500` (深橙)
- 字体：Space Grotesk (标题) + Plus Jakarta Sans (正文)
- 像素风立体阴影效果

## 使用方式

1. 点击菜单栏图标 → 选择「选项」
2. 拖动滑块调整桌宠大小
3. 点击「RESET」重置为 100%
4. 点击「APPLY & CLOSE」保存并关闭

## 注意事项

- 缩放以猫咪底部为中心点
- 响应状态（大气泡）和工作状态（小气泡）会跟随缩放
- 切换 Response/Working 状态时窗口尺寸会正确调整
