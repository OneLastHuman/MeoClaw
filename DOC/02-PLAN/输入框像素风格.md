# 输入框像素风格

## 概述

将桌宠的输入框从原来的 `Input.png` 图片背景替换为纯 CSS 实现的像素风格控件 `pixel-bar`，并接入当前的统一缩放模型。

## 设计特点

### 像素风格边框
- 多层 inset box-shadow 实现像素风格的立体边框效果
- 4px 黑色外边框
- 浅灰色和深灰色 inset 层模拟凹陷效果
- 左上角白色高光层，右下角深灰色阴影层

### 布局参数
| 参数 | 值 | 说明 |
|------|-----|------|
| inputWidth | 160 | 输入框总宽度 |
| inputHeight | 44 | 输入框总高度 |
| inputFontSize | 18 | 文字大小 |
| inputStep | 3 | 像素单位（用于 box-shadow 计算） |

### 组件结构
```
.pixel-bar (容器)
├── .pixel-input (文本输入框)
└── .mic-btn (麦克风按钮)
```

### CSS 变量
在 `.pixel-bar` 内定义的本地变量：
- `--p-black`: #000000 (黑色边框)
- `--p-white`: #ffffff (白色背景)
- `--p-gray-light`: #bcbcbc (浅灰层)
- `--p-gray-dark`: #818181 (深灰阴影层)
- `--p-bg`: #e7e7e7 (页面背景，用于角遮罩)
- `--step`: var(--input-step) (像素单位)

## 样式细节

### pixel-bar 容器
- 使用多层 box-shadow 实现像素边框效果
- `::before` 伪元素创建四角背景色遮罩，实现像素转角效果
- flex 布局，左侧输入框，右侧麦克风按钮

### pixel-input 输入框
- `flex: 1` 占据剩余空间
- 无边框、无轮廓，透明背景
- placeholder 文字：`ask me anything...`
- 字体：monospace

### mic-btn 麦克风按钮
- 宽度 26px，高度适应输入框
- 背景色为浅灰色，带有 inset 立体效果
- 悬停时背景加深
- 点击时阴影反转模拟按下效果
- 图标改为内联 SVG，避免额外 CDN 依赖

## 与当前缩放实现的关系

- 缩放参数统一定义在 `src/stores/petSettings.ts`
- `DesktopPet.vue` 通过 `rootCSSVars` 注入：
  - `--input-width`
  - `--input-height`
  - `--input-font-size`
  - `--input-step`
  - `--pet-scale`
- 因此输入框在 `50% / 100% / 200%` 下会和桌宠本体同步缩放，不再依赖固定像素图资源
