# 使用 CrabNebula Cloud 进行分发

_Source: https://v2.tauri.org.cn/distribute/crabnebula-cloud/_

[CrabNebula](https://crabnebula.dev) 是 Tauri 的官方合作伙伴，为 Tauri 应用程序提供服务和工具。[CrabNebula Cloud](https://crabnebula.dev/cloud/) 是一个与 Tauri 更新程序无缝集成的应用程序分发平台。

该云平台提供内容分发网络 (CDN)，能够在全球范围内高效且经济地发布您的应用程序安装程序和更新，并提供下载指标数据。

借助 CrabNebula Cloud 服务，您可以轻松实现多个发布渠道、为您的应用程序网站添加下载按钮等功能。

设置您的 Tauri 应用以使用该云服务非常简单：您只需使用 GitHub 账号登录 [Cloud 网站](https://web.crabnebula.cloud/)，创建您的组织和应用程序，然后安装其 CLI 以创建发布并上传 Tauri 软件包。此外，平台还提供了 [GitHub Action](https://github.com/crabnebula-dev/cloud-release/)，以简化在 GitHub 工作流中使用该 CLI 的过程。

如需了解更多信息，请参阅 [CrabNebula Cloud 文档](https://docs.crabnebula.dev/cloud/)。
