## 使用 CrabNebula Cloud 进行分发

标题为“使用 CrabNebula Cloud 进行分发”的章节

[CrabNebula](https://crabnebula.dev) 是 Tauri 的官方合作伙伴，为 Tauri 应用提供服务和工具。[CrabNebula Cloud](https://crabnebula.dev/cloud/) 是一个与 Tauri 更新程序无缝集成的应用分发平台。

Cloud 提供的内容分发网络 (CDN) 能够以经济高效的方式在全球范围内分发您的应用安装程序和更新，并提供下载指标。

通过 CrabNebula Cloud 服务，您可以轻松实现多个发布渠道、为您的应用网站添加下载按钮等功能。

设置您的 Tauri 应用以使用 Cloud 非常简单：只需使用 GitHub 账号登录 [Cloud 网站](https://web.crabnebula.cloud/)，创建您的组织和应用，并安装其 CLI 以创建发布版本并上传 Tauri 捆绑包即可。此外，还提供了一个 [GitHub Action](https://github.com/crabnebula-dev/cloud-release/)，以简化在 GitHub 工作流中使用 CLI 的过程。

如需了解更多信息，请参阅 [CrabNebula Cloud 文档](https://docs.crabnebula.dev/cloud/)。
