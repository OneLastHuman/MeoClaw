# 桌面宠物应用架构参考

## 概述

桌面宠物（Desktop Pet / Companion）是一类特殊的桌面应用程序，具有以下特点：
- 常驻桌面，窗口置顶
- 轻量交互，快速响应
- 可视化形象（2D/3D）
- 多窗口架构（主窗口 + 气泡/信息窗口）

## 架构模式

### 1. 单窗口模式

所有内容在一个窗口内，通过 CSS/JS 切换视图。

**优点**: 简单，通信容易
**缺点**: 功能扩展性差

### 2. 多窗口模式（推荐）

```
┌─────────────────┐      ┌─────────────────┐
│   Main Window   │      │  Bubble Window  │
│   (Pet Sprite)  │ ──── │  (Info/Panel)   │
│   180×180       │ 通信  │  380×560        │
└─────────────────┘      └─────────────────┘
        │                        │
        └────────┬───────────────┘
                 ▼
        ┌─────────────────┐
        │  Rust Backend   │
        │  (State Center) │
        └─────────────────┘
```

### 3. WebView + Native Hybrid

前端使用 Web 技术（Vue/React），Native 功能（窗口管理、系统集成）由 Rust/Tauri 处理。

## 窗口职责划分

### 主窗口 (Main Window)

| 属性 | 值 |
|------|-----|
| 尺寸 | 180×180 |
| 特性 | 透明背景、无边框、置顶、点击穿透（可选） |
| 职责 | 显示宠物精灵、响应点击事件、触发交互 |

### 气泡窗口 (Bubble Window)

| 属性 | 值 |
|------|-----|
| 尺寸 | 380×560 |
| 特性 | 透明背景、无边框、跟随主窗口、置顶 |
| 职责 | 显示工作状态、输入框、消息队列 |

### 实现要点

```rust
// 创建主窗口
let main_window = WebviewWindowBuilder::new(app, "main", WebviewUrl::App("index.html".into()))
    .title("Desktop Pet")
    .inner_size(180.0, 180.0)
    .decorations(false)         // 无边框
    .transparent(true)          // 透明背景
    .always_on_top(true)        // 置顶
    .skip_taskbar(true)         // 不显示在任务栏
    .resizable(false)           // 不可缩放
    .build()?;

// 创建气泡窗口
let bubble_window = WebviewWindowBuilder::new(app, "bubble", WebviewUrl::App("bubble.html".into()))
    .inner_size(380.0, 560.0)
    .decorations(false)
    .transparent(true)
    .always_on_top(true)
    .skip_taskbar(true)
    .visible(false)             // 默认隐藏
    .build()?;
```

## 窗口间通信

### 方案一：Rust 后端转发

```
Pet Window ──▶ Rust Backend ──▶ Bubble Window
     │              ▲
     │              │
     └──────────────┘
         (状态更新)
```

```rust
// Rust: 监听主窗口事件，发送到气泡窗口
#[tauri::command]
fn show_bubble(app: AppHandle, content: String) {
    app.emit_to("bubble", "show-content", content).unwrap();
}

// Rust: 气泡跟随主窗口
app.listen(WindowEvent::Moved(position), |event| {
    // 计算气泡位置并移动
});
```

### 方案二：前端直连（不推荐）

前端窗口之间不应该直接通信，应通过 Rust 后端中转。

```typescript
// ❌ 不推荐：直接访问其他窗口
const bubble = getWebviewWindow('bubble');
await bubble.emit('event', data);

// ✓ 推荐：通过 Rust 转发
await invoke('route_to_bubble', { event: 'show', data });
```

## 状态管理架构

### 中心化状态 vs 分布式状态

| 方案 | 适用场景 | 优点 | 缺点 |
|------|----------|------|------|
| **中心化** | 多窗口同步 | 状态一致性强 | 单点瓶颈 |
| **分布式** | 独立窗口 | 解耦 | 同步复杂 |

### 推荐：中心化状态 + 事件通知

```rust
// 状态存储（Rust）
struct AppState {
    current_animation: Mutex<String>,
    bubble_content: Mutex<String>,
    window_positions: Mutex<HashMap<String, Position>>,
}

// 状态查询
#[tauri::command]
fn get_state(state: State<AppState>) -> AppStateSnapshot {
    AppStateSnapshot {
        animation: state.current_animation.lock().unwrap().clone(),
        bubble: state.bubble_content.lock().unwrap().clone(),
    }
}
```

### 前端状态管理

```typescript
// Vue 3 + Pinia 示例
import { defineStore } from 'pinia';

export const usePetStore = defineStore('pet', {
    state: () => ({
        currentState: 'idle' as AnimState,
        bubbleVisible: false,
        bubbleContent: '',
    }),
    
    actions: {
        async switchAnimation(state: AnimState) {
            this.currentState = state;
            await invoke('switch_animation', { state });
        },
        
        async showBubble(content: string) {
            this.bubbleVisible = true;
            this.bubbleContent = content;
            await invoke('show_bubble', { content });
        },
    },
});
```

## 文件监控模式

适合需要响应外部文件变化的桌面宠物应用。

### watchdeps 库

```rust
use notify::{Config, RecommendedWatcher, RecursiveMode, Watcher};
use std::path::Path;
use std::sync::mpsc;
use std::time::Duration;

fn start_file_watcher(app: AppHandle, path: String) {
    std::thread::spawn(move || {
        let (tx, rx) = mpsc::channel();
        
        let mut watcher = RecommendedWatcher::new(tx).unwrap();
        watcher.watch(Path::new(&path), RecursiveMode::NonRecursive).unwrap();
        
        loop {
            match rx.recv_timeout(Duration::from_millis(500)) {
                Ok(event) => {
                    if event.kind.is_modify() {
                        // 文件变化，触发状态更新
                        app.emit("file-changed", path.clone()).unwrap();
                    }
                }
                Err(mpsc::RecvTimeoutError::Timeout) => continue,
                Err(_) => break,
            }
        }
    });
}
```

## 性能优化

### 1. 减少重绘

- 使用 `will-change` 提示浏览器
- 避免频繁的 DOM 操作
- 使用 CSS transform 替代 top/left

```css
.pet-sprite {
    transform: translateZ(0);
    will-change: background-position;
}
```

### 2. 节流/防抖

```typescript
// 节流：限制事件频率
function throttle<T extends (...args: any[]) => void>(
    fn: T,
    delay: number
): T {
    let lastCall = 0;
    return ((...args: any[]) => {
        const now = Date.now();
        if (now - lastCall >= delay) {
            lastCall = now;
            fn(...args);
        }
    }) as T;
}

// 使用：窗口移动事件（每 50ms 最多一次）
const onMovedThrottled = throttle((pos: Position) => {
    updateBubblePosition(pos);
}, 50);
```

### 3. 预加载资源

```typescript
class ResourcePreloader {
    async preloadAll() {
        const images = [
            '/anim/idle.png',
            '/anim/working.png',
            '/anim/shock.png',
            // ...
        ];
        
        await Promise.all(
            images.map(src => {
                return new Promise((resolve, reject) => {
                    const img = new Image();
                    img.onload = resolve;
                    img.onerror = reject;
                    img.src = src;
                });
            })
        );
    }
}
```

## 系统集成

### 1. 右键菜单

使用 Tauri Menu API：

```rust
use tauri::menu::{Menu, MenuItem};

#[tauri::command]
async fn show_context_menu(app: AppHandle) -> Result<(), String> {
    let quit = MenuItem::with_id(app, "quit", "Quit", true, None::<&str>).map_err(|e| e.to_string())?;
    let menu = Menu::with_items(app, &[&quit]).map_err(|e| e.to_string())?;
    
    menu.popup().map_err(|e| e.to_string())?;
    Ok(())
}
```

### 2. 系统托盘

```rust
use tauri::menu::{Menu, MenuItem};
use tauri::tray::{MouseButton, MouseButtonState, TrayIconBuilder, TrayIconEvent};

fn setup_tray(app: &App) -> Result<(), Box<dyn std::error::Error>> {
    let quit = MenuItem::with_id(app, "quit", "Quit", true, None::<&str>)?;
    let tray_menu = Menu::with_items(app, &[&quit])?;
    
    TrayIconBuilder::new()
        .menu(&tray_menu)
        .on_menu_event(|app, event| {
            if event.id.as_ref() == "quit" {
                app.exit(0);
            }
        })
        .build(app)?;
    
    Ok(())
}
```

### 3. 全局快捷键

```rust
use tauri::GlobalShortcutManager;

app.global_shortcut_manager().register("CmdOrCtrl+Shift+P", || {
    // 显示/隐藏宠物
})?;
```
