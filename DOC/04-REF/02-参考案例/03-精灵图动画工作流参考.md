# 精灵图动画工作流参考

## 概述

精灵图（Sprite Sheet）动画是 2D 游戏和桌面宠物应用中最常用的动画技术。通过预先绘制好的帧图像序列，快速切换显示形成动画效果。

## 核心概念

### 精灵图结构

```
┌────┬────┬────┬────┐
│ 0  │ 1  │ 2  │ 3  │  ← Row 0
├────┼────┼────┼────┤
│ 4  │ 5  │ 6  │ 7  │  ← Row 1
├────┼────┼────┼────┤
│ 8  │ 9  │ 10 │ 11 │  ← Row 2
├────┼────┼────┼────┤
│ 12 │ 13 │ 14 │ 15 │  ← Row 3
└────┴────┴────┴────┘
  col=4, row=4, total=16 frames
```

### 精灵图规格参数

| 参数 | 说明 | 示例 |
|------|------|------|
| `cols` | 列数 | 4 |
| `rows` | 行数 | 4 |
| `totalFrames` | 总帧数 | 16 |
| `frameWidth` | 单帧宽度 | 180px |
| `frameHeight` | 单帧高度 | 180px |
| `interval` | 帧间隔 | 100ms |

### 帧索引计算

```typescript
function getFramePosition(frameIndex: number, cols: number) {
    const col = frameIndex % cols;      // 列索引
    const row = Math.floor(frameIndex / cols);  // 行索引
    return { col, row };
}

// 示例：帧 7 → col=3, row=1
// 背景位置: -3*frameWidth, -1*frameHeight
```

## CSS Sprite 动画实现

### 基础实现

```css
.sprite {
    width: 180px;
    height: 180px;
    background-image: url('/anim/cat_idle.png');
    background-repeat: no-repeat;
    /* 精灵图总尺寸: 720x720 (4x4) */
    background-size: 720px 720px;
}

@keyframes play-idle {
    0%   { background-position: 0 0; }
    25%  { background-position: -180px 0; }
    50%  { background-position: -360px 0; }
    75%  { background-position: -540px 0; }
    100% { background-position: 0 0; }
}
```

### 动态计算（推荐）

```typescript
function getBackgroundPosition(frame: number, frameSize: number, cols: number) {
    const x = -(frame % cols) * frameSize;
    const y = -Math.floor(frame / cols) * frameSize;
    return `${x}px ${y}px`;
}

// 使用
element.style.backgroundPosition = getBackgroundPosition(7, 180, 4);
// 结果: "-540px -180px"
```

## Canvas 绘制方式

### 单帧绘制

```javascript
const canvas = document.getElementById('sprite-canvas');
const ctx = canvas.getContext('2d');

const sprite = new Image();
sprite.src = '/anim/cat_idle.png';

sprite.onload = () => {
    function drawFrame(frameIndex) {
        const cols = 4;
        const frameSize = 180;
        
        const col = frameIndex % cols;
        const row = Math.floor(frameIndex / cols);
        const sx = col * frameSize;
        const sy = row * frameSize;
        
        ctx.clearRect(0, 0, canvas.width, canvas.height);
        ctx.drawImage(
            sprite,
            sx, sy, frameSize, frameSize,  // 源矩形
            0, 0, canvas.width, canvas.height  // 目标矩形
        );
    }
    
    // 动画循环
    let currentFrame = 0;
    setInterval(() => {
        drawFrame(currentFrame);
        currentFrame = (currentFrame + 1) % 16;
    }, 100);
};
```

## 动画配置数据结构

### 推荐配置格式

```typescript
interface AnimationConfig {
    name: string;              // 动画名称
    cols: number;              // 精灵图列数
    rows: number;             // 精灵图行数
    totalFrames: number;       // 总帧数
    interval: number;          // 帧间隔(ms)
    loop: boolean;             // 是否循环
    pingpong?: boolean;        // 是否来回播放
    startFrame?: number;       // 起始帧索引（用于部分帧）
    imagePath: string;         // 精灵图路径
    nextState?: AnimState;     // 播放完后目标状态
}

interface Animations {
    [stateName: string]: AnimationConfig;
}
```

### 示例配置

```typescript
const ANIMATIONS: Animations = {
    idle: {
        name: 'idle',
        cols: 4,
        rows: 4,
        totalFrames: 16,
        interval: 100,
        loop: true,
        startFrame: 0,
        imagePath: '/anim/idle.png',
    },
    
    working: {
        name: 'working',
        cols: 7,
        rows: 7,
        totalFrames: 35,        // 帧 13-47
        interval: 100,
        loop: true,
        pingpong: false,
        startFrame: 13,         // 从第 13 帧开始
        imagePath: '/anim/work.png',
        nextState: 'idle',      // 可选：播放完切换
    },
    
    shock: {
        name: 'shock',
        cols: 3,
        rows: 4,
        totalFrames: 10,
        interval: 150,
        loop: false,
        startFrame: 0,
        imagePath: '/anim/shock.png',
        nextState: 'idle',      // 单次播放后回 idle
    },
};
```

## 精灵图生成工作流

### 从动画软件导出

1. **Aseprite** - 像素艺术专用
   - 导出为 `frames.png`（垂直排列）
   - 支持动画预览和帧编辑

2. **TexturePacker** - 通用纹理打包
   - 自动排布精灵图
   - 生成 JSON 配置

3. **自定义脚本**

```python
#!/usr/bin/env python3
"""精灵图生成脚本"""
from PIL import Image
import os

def create_sprite_sheet(frames_dir, output_path, cols=4):
    """将帧图像合并为精灵图"""
    frames = sorted([f for f in os.listdir(frames_dir) if f.endswith('.png')])
    
    if not frames:
        raise ValueError("No frames found")
    
    # 读取第一帧获取尺寸
    first = Image.open(os.path.join(frames_dir, frames[0]))
    frame_width, frame_height = first.size
    
    # 计算网格
    rows = (len(frames) + cols - 1) // cols
    
    # 创建画布
    sheet = Image.new('RGBA', (frame_width * cols, frame_height * rows))
    
    # 填充帧
    for i, frame_file in enumerate(frames):
        img = Image.open(os.path.join(frames_dir, frame_file))
        x = (i % cols) * frame_width
        y = (i // cols) * frame_height
        sheet.paste(img, (x, y))
    
    sheet.save(output_path)
    print(f"Created sprite sheet: {output_path} ({cols}x{rows})")

if __name__ == '__main__':
    create_sprite_sheet('frames/idle', 'public/anim/idle.png')
```

## 性能优化

### 1. 预加载精灵图

```typescript
class SpriteLoader {
    private images: Map<string, HTMLImageElement> = new Map();
    
    async preload(imagePaths: string[]): Promise<void> {
        await Promise.all(
            imagePaths.map(path => {
                return new Promise<void>((resolve, reject) => {
                    const img = new Image();
                    img.onload = () => {
                        this.images.set(path, img);
                        resolve();
                    };
                    img.onerror = reject;
                    img.src = path;
                });
            })
        );
    }
    
    getImage(path: string): HTMLImageElement | undefined {
        return this.images.get(path);
    }
}
```

## 动画状态与状态机集成

### 关键设计原则

1. **状态决定动画**: 动画状态由游戏/应用状态机决定
2. **动画驱动状态转换**: 某些动画播放完后自动切换状态（单次动画）
3. **可中断性**: 高优先级状态（shock）可中断当前动画

### 集成示例

```typescript
class AnimatedCharacter {
    private stateMachine: StateMachine;
    private sprite: SpriteRenderer;
    
    update(deltaTime: number) {
        const currentState = this.stateMachine.currentState;
        const animConfig = ANIMATIONS[currentState];
        
        if (this.sprite.isAnimationFinished()) {
            if (animConfig.nextState) {
                this.stateMachine.transition(animConfig.nextState);
            }
        }
        
        this.sprite.play(animConfig);
    }
}
```

## 11. OffscreenCanvas + Web Worker 渲染

Canvas 在主线程渲染，主线程繁忙时会掉帧。使用 Worker 进行渲染可保持 60fps。

### 架构

```
Main Thread          Web Worker
┌──────────┐        ┌──────────┐
│ 状态机   │ ──────▶│ 渲染计算 │
│ 物理引擎 │        │ 精灵图绘制│
│ DOM更新  │ ◀──────│ 位图发送 │
└──────────┘        └──────────┘
```

### Worker 实现

```typescript
// render.worker.ts
const canvas = new OffscreenCanvas(180, 180);
const ctx = canvas.getContext('2d')!;

self.onmessage = (e) => {
    const { type, data } = e.data;
    
    if (type === 'LOAD_SPRITE') {
        // 加载精灵图
        const bitmap = await createImageBitmap(fetch(data.url));
        self.postMessage({ type: 'SPRITE_LOADED', bitmap }, [bitmap]);
    }
    
    if (type === 'RENDER') {
        drawFrame(data.frame);
        const bitmap = canvas.transferToImageBitmap();
        self.postMessage({ type: 'FRAME', bitmap }, [bitmap]);
    }
};
```

## 12. 物理引擎集成 (Matter.js)

桌面宠物的"真实感"需要重力、惯性、碰撞。

### 安装

```bash
npm install matter-js
```

### 基础用法

```typescript
import Matter from 'matter-js';

const engine = Matter.Engine.create({ gravity: { y: 0.5 } });
const cat = Matter.Bodies.circle(400, 200, 50, {
    restitution: 0.6,  // 弹性
    friction: 0.3,
});
Matter.World.add(engine.world, cat);

// 鼠标拖拽
Matter.MouseConstraint.create(engine, {
    onEnd: (e) => {
        // 释放时的速度 = 抛出速度
        Matter.Body.setVelocity(cat, e.mouse.position);
    }
});
```

### 与精灵图同步

```typescript
Matter.Events.on(engine, 'afterUpdate', () => {
    const { x, y } = cat.position;
    const angle = cat.angle;
    // 更新 Canvas 位置
    ctx.setTransform(1, 0, 0, 1, x - 90, y - 90);
    ctx.rotate(angle);
});
```
