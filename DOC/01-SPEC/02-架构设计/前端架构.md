# 前端架构

## 目录结构

```
src/
├── App.vue                 # 根组件
├── main.ts                 # 应用入口
├── components/
│   └── DesktopPet.vue      # 主桌宠组件
├── config/
│   └── animations.ts       # 动画状态机配置
├── stores/
│   └── petSettings.ts      # 宠物设置状态（Vue Composition API）
├── menu/
│   └── contextMenu.ts     # 右键菜单
└── windowBehavior.ts       # 窗口边缘吸附行为
```

## 技术栈

- **框架**: Vue 3 Composition API
- **状态管理**: Vue `ref` + `computed` + `watch`（非 Pinia）
- **构建工具**: Vite
- **UI**: 原生 CSS（像素风格）

## 核心组件

### DesktopPet.vue

主桌宠组件，负责:
- **动画渲染**: 精灵图动画系统
- **用户交互**: 点击、双击、拖拽
- **输入处理**: 文本输入、文件拖拽
- **窗口行为**: 边缘隐藏/吸附/预览
- **事件监听**: 监听 Tauri 事件更新 UI

**动画状态 (AnimState)**:

```typescript
type AnimState =
  | 'idle'           // 空闲
  | 'EnterInput'      // 进入输入模式
  | 'shock'           // 惊讶
  | 'startworking'    // 开始工作
  | 'working'         //工作中
  | 'workingPreview'  // 工作预览
  | 'EnterReceiving'  // 进入接收模式
  | 'Receiving'       // 接收中
  | 'received'        // 已接收
  | 'Response';       // 响应显示
```

**精灵图动画系统**:

- 通过 CSS `background-position` 切换帧
- 支持循环/非循环/ping-pong 模式
- 配置化设计 (animations.ts)

### petSettings.ts

使用 Vue `ref` + `watch` 实现响应式配置:

```typescript
// 缩放配置
export const petScale = ref(loadPetScale());
watch(petScale, persistPetScale);

// 响应音效配置
export const responseSound = ref(loadResponseSound());
watch(responseSound, persistResponseSound);
```

**配置项**:

| 配置 | 存储 | 默认值 |
|------|------|--------|
| petScale | localStorage | 1 |
| responseSound | localStorage | 'chime.mp3' |

### animations.ts

动画配置定义:

```typescript
export interface AnimationConfig {
  name: string;        // 动画名称
  cols: number;        // 精灵图列数
  rows: number;        // 精灵图行数
  totalFrames: number; // 总帧数
  interval: number;    // 帧间隔 (ms)
  loop: boolean;       // 是否循环
  pingpong?: boolean;  // 来回播放
  startFrame: number;  // 起始帧
  imagePath: string;   // 精灵图路径
  nextState?: AnimState; // 播放完毕后跳转状态
}
```

### windowBehavior.ts

边缘吸附行为状态机:

```typescript
type WindowBehaviorState =
  | 'normal'         // 正常显示
  | 'dragging'       // 拖拽中
  | 'edgeHiddenLeft'    // 隐藏到左侧
  | 'edgeHiddenRight'   // 隐藏到右侧
  | 'edgePeekLeft'      // 左侧预览
  | 'edgePeekRight';    // 右侧预览
```

**行为模式**:

| 模式 | 描述 |
|------|------|
| compact | 紧凑模式，隐藏到边缘 |
| hidden | 完全隐藏 |
| peek | 悬停预览 |

## 事件系统

### Tauri 事件监听

```typescript
// 动画状态切换
await listen<string>('switch-animation', (event) => {
  switchAnim(event.payload as AnimState);
});

// 工具调用气泡更新
await listen<BubblePayload>('update-bubble', (event) => {
  bubbleTools.value = event.payload.tools;
});

// 响应气泡更新
await listen<ResponseBubblePayload>('update-response-bubble', (event) => {
  responseBubbleText.value = event.payload.text;
});
```

### 事件发射 (invoke)

```typescript
// 切换动画
await invoke('switch_animation', { state: 'working' });

// 发送消息
await invoke('send_message', { message: text, attachments: [] });

// 窗口操作
await invoke('show_bubble_window', { mainX, mainY });
await invoke('enter_response_mode');
```

## 窗口尺寸管理

根据动画状态动态调整窗口:

| 状态 | 窗口尺寸 | 布局 |
|------|----------|------|
| normal | 180x180 | 仅桌宠 |
| working | 320x~250 | 桌宠 + 气泡 |
| response | ~480x~650+ | 桌宠 + 响应气泡 |

**底部锚定算法**: 保持猫咪底部位置不变，通过 `resizeWindowPreservingBottomLeft` 实现。

## 样式系统

- **CSS 变量**: 基于 `--pet-scale` 的响应式缩放
- **像素风 UI**: 8-bit 风格输入框
- **气泡样式**: Markdown 渲染支持 (表格、代码块、引用等)
