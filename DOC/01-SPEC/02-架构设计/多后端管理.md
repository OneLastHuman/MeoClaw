# 多后端管理

## 概述

MeoClaw 支持同时连接多个 AI 后端，通过 `BackendManager` 统一管理后端客户端的注册、切换和健康检查。

## 支持的后端

| 后端 | 类型 | 协议 |
|------|------|------|
| OpenClaw | WebSocket | 自动发现（18789 → 19001） |
| Hermes | HTTP/SSE | http://127.0.0.1:8642 |

## BackendManager

### 核心结构

```rust
pub struct BackendManager {
    config: Arc<Mutex<BackendConfig>>,
    clients: Arc<Mutex<HashMap<String, Box<dyn BackendClient>>>>,
}
```

### 配置结构

```rust
pub struct BackendConfig {
    pub selected: ActiveBackend,           // 当前选中的后端
    pub openclaw: OpenClawConfig,          // OpenClaw 配置
    pub hermes: LegacyHermesConfig,        // Hermes 配置（兼容旧接口）
    pub discovered_instances: Vec<HermesConfig>,  // 发现的所有 Hermes 实例
}
```

### 活跃后端枚举

```rust
pub enum ActiveBackend {
    OpenClaw,
    Hermes,
}
```

## 客户端注册

```rust
// 注册后端客户端
manager.register_client(Box::new(OpenClawClientAdapter::new()));
manager.register_client(Box::new(hermes_client));
```

## 消息路由

```rust
pub fn send_message_to_client(
    &self,
    name: &str,
    message: &str,
) -> Result<Box<dyn Stream>, String>
```

根据当前选中的后端自动路由消息：
- OpenClaw：使用 WebSocket 发送
- Hermes：使用 HTTP POST 发送

## 后端切换

```rust
pub fn switch(&self, backend: &str) -> Result<(), String>
```

切换当前活跃后端，同时更新配置文件。

## 健康检查

```rust
pub fn check_health(&self) -> BackendHealth
```

返回结构：
```rust
pub struct BackendHealth {
    pub available: bool,       // 是否可用
    pub backend: String,       // 后端名称
    pub endpoint: String,      // 端点地址
    pub error: Option<String>, // 错误信息
}
```

## 配置文件

**路径**: `~/.local/share/meo-claw/backend-config.json`

**结构**:
```json
{
  "selected": "openclaw",
  "openclaw": {
    // endpoint 由运行时端口发现机制确定（18789 或 19001）
    "endpoint": null,
    "health_endpoint": null
  },
  "hermes": {
    "endpoint": "http://127.0.0.1:8642",
    "health_endpoint": "http://127.0.0.1:8642/health"
  },
  "discovered_instances": []
}
```

## Tauri 命令

| 命令 | 说明 |
|------|------|
| `switch_backend` | 切换后端 |
| `get_current_backend` | 获取当前后端 |
| `check_backend_health` | 健康检查（当前选中后端） |
| `check_all_backends_health` | 健康检查（所有已注册后端） |

## 托盘菜单集成

通过系统托盘菜单可以快速切换后端：

```
┌─────────────────┐
│ 选项            │
├─────────────────┤
│ 选择后端        │
│  ├─ OpenClaw   │
│  └─ Hermes     │
│  └─ 检查状态   │
├─────────────────┤
│ 重启            │
│ 退出            │
└─────────────────┘
```
