# 窗口通信

本文档描述 MeoClaw 应用中多窗口之间的通信方式。

## 窗口架构

```
┌──────────────────────────────────────────────────────────────┐
│                      主窗口 (main)                           │
│  尺寸: 180x180 (正常) / 1120x1480 (响应模式)                 │
│  内容: DesktopPet.vue (桌宠主界面)                            │
└──────────────────────────────────────────────────────────────┘
           │                           │
           │  跟随移动                  │ 响应时扩展
           ▼                           ▼
┌──────────────────────┐    ┌──────────────────────────────┐
│   气泡窗口 (bubble)   │    │   选项窗口 (options)          │
│   尺寸: 380x560       │    │   尺寸: 680x550               │
│   内容: bubble.html   │    │   内容: options.html          │
│   用途: 工具气泡/响应  │    │   用途: 后端配置              │
└──────────────────────┘    └──────────────────────────────┘
```

## 窗口配置

### 主窗口 (main)

**创建方式**: Tauri 默认主窗口

**尺寸**:
- 正常模式: `360x360` (实际显示 180x180 桌宠)
- 响应模式: `1120x1480` (扩展显示响应内容)

**特性**:
- 透明背景
- 无装饰框
- 置顶显示
- 可拖拽

### 气泡窗口 (bubble)

**创建方式**: `WebviewWindowBuilder`

**位置**: `src-tauri/src/lib.rs:538`

```rust
let bubble_window = WebviewWindowBuilder::new(
    &app,
    "bubble",
    WebviewUrl::App("bubble.html".into()),
)
.title("Bubble")
.inner_size(380.0, 560.0)
.decorations(false)      // 无装饰
.transparent(true)       // 透明背景
.always_on_top(true)     // 置顶
.skip_taskbar(true)      // 不显示在任务栏
.shadow(false)           // 无阴影
.resizable(false)        // 不可调整大小
.visible(true)
.build()
```

### 选项窗口 (options)

**创建方式**: `show_options_window` 函数

**位置**: `src-tauri/src/lib.rs:640`

```rust
let options_window = WebviewWindowBuilder::new(
    app,
    "options",
    WebviewUrl::App("options.html".into()),
)
.title("MeoClaw 选项")
.inner_size(680.0, 550.0)
.min_inner_size(620.0, 480.0)
.resizable(false)
.maximizable(false)
.minimizable(false)
.always_on_top(true)
.skip_taskbar(true)
.decorations(false)      // 无边框
.transparent(true)       // 透明背景（支持圆角）
.build()
```

**特性**:
- 无边框设计，通过 `data-tauri-drag-region` 在 title bar 区域实现窗口拖拽
- 标题栏左侧有红色圆形关闭按钮，右侧居中显示 "MEOCLAW OPTIONS"
- 窗口背景透明，内容区使用 `border-radius: 12px` 显示圆角

## 窗口间位置关系

### 气泡窗口定位

气泡窗口位于主窗口右上角，具体算法：

```
气泡左下角 = 主窗口右上角

计算公式:
bubble_x = main_x + 180        (主窗口宽度)
bubble_y = main_y - 560        (气泡高度)

即: 气泡的左上角 = (main_x + 180, main_y - 560)
```

### 跟随移动

**方式 1**: 主窗口移动事件监听 (`lib.rs:1033`)

```rust
main_window.on_window_event(move |event| {
    if let tauri::WindowEvent::Moved(position) = event {
        let new_x = position.x + 180;
        let new_y = position.y as i32 - 560;
        bubble.set_position(...);
    }
});
```

**方式 2**: `move_bubble_window` 命令

由 Frontend 在特定场景下主动调用。

### 响应模式窗口调整

**进入响应模式** (`enter_response_mode`):

```
原始: 360x360, 猫咪底部在 y = pos.y + 180
目标: 1120x1480, 猫咪底部保持在相同 y

算法:
1. 先 resize: 360x360 → 1120x1480 (macOS 向下扩展)
2. 再移动: 保持 cat_y 不变
   target_y = cat_y_before - 1300
```

**退出响应模式** (`exit_response_mode`):

```
原始: 1120x1480, 猫咪底部在 y = pos.y + 180
目标: 360x360, 猫咪底部保持在相同 y

算法:
1. 先 resize: 1120x1480 → 360x360 (macOS 向上收缩)
2. 再移动: 保持 cat_y 不变
   target_y = cat_y_before - 180
```

## 窗口间通信

### Rust → Frontend (通过 emit)

所有窗口间通信通过 Tauri 的事件系统：

```rust
// 发送到所有窗口
app.emit("event-name", payload)

// 发送到特定窗口
app.get_webview_window("bubble")?.emit("event-name", payload)
```

### Frontend → Rust (通过 invoke)

Frontend 主动调用 Rust 命令：

```typescript
import { invoke } from '@tauri-apps/api/core';

// 显示气泡窗口（需要传递主窗口位置）
await invoke('show_bubble_window', {
    main_x: position.x,
    main_y: position.y
});

// 移动气泡窗口
await invoke('move_bubble_window', {
    main_x: position.x,
    main_y: position.y
});

// 进入/退出响应模式
await invoke('enter_response_mode');
await invoke('exit_response_mode');
```

## 窗口生命周期

### 气泡窗口

| 阶段 | 操作 | 说明 |
|-----|------|------|
| 显示 | `show_bubble_window` | 如果已存在则直接显示并移动 |
| 隐藏 | `hide_bubble_window` | 隐藏窗口（不销毁） |
| 移动 | 跟随主窗口 | 通过 `on_window_event` 自动跟随 |
| 销毁 | 应用退出 | 窗口自然关闭 |

### 选项窗口

| 阶段 | 操作 | 说明 |
|-----|------|------|
| 打开 | 托盘菜单 → 选项 | 调用 `show_options_window()` |
| 显示 | 已有窗口时 | 调用 `show()` + `set_focus()` |
| 关闭 | 用户关闭 | 窗口关闭（不退出应用） |

## 前端窗口操作

### DesktopPet.vue

```typescript
// 显示气泡窗口
const showBubble = async () => {
    const position = await getWindowPosition();
    await invoke('show_bubble_window', {
        main_x: position.x,
        main_y: position.y
    });
};

// 隐藏气泡窗口
const hideBubble = async () => {
    await invoke('hide_bubble_window');
};

// 进入响应模式
await invoke('enter_response_mode');

// 退出响应模式
await invoke('exit_response_mode');
```

### options.ts

```typescript
// 切换后端
await invoke('switch_backend', { backend });

// 获取当前后端
const backend = await invoke<string | null>('get_current_backend');

// 检查所有后端健康状态（同时更新 OpenClaw 和 Hermes 的指示器）
const healthList = await invoke<BackendHealth[]>('check_all_backends_health');
healthList.forEach(health => {
  const indicator = document.getElementById(`${health.backend}-indicator`);
  const textEl = document.getElementById(`${health.backend}-text`);
  if (indicator) {
    indicator.className = `selector-indicator ${health.available ? 'running' : 'stopped'}`;
  }
  if (textEl) {
    textEl.textContent = health.available ? 'ON' : 'OFF';
  }
});
```

## 托盘菜单窗口操作

**托盘图标左键**: 切换主窗口显示/隐藏

```rust
.on_tray_icon_event(move |_tray, event| {
    if let TrayIconEvent::Click { button: MouseButton::Left, ... } = event {
        if let Some(window) = main_win.get_webview_window("main") {
            if window.is_visible().unwrap_or(false) {
                window.hide();
            } else {
                window.show();
            }
        }
    }
})
```

**托盘菜单 → 选项**: 打开选项窗口

```rust
"options" => {
    show_options_window(app)?;
}
```

## 窗口状态持久化

通过 localStorage 持久化：

```typescript
// 宠物缩放
localStorage.setItem('meoclaw.petScale', scale.toString());
window.addEventListener('storage', (event) => {
    if (event.key === 'meoclaw.petScale') {
        setPetScale(Number(event.newValue));
    }
});
```

## 备注

- 所有窗口共享同一个 Rust 后端实例
- 事件通过 `app.emit()` 发送到所有窗口
- 气泡窗口与主窗口通过相对位置保持对齐
- 选项窗口为一次性窗口，关闭后重新打开会创建新实例
