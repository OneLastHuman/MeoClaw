## 各状态显示模板

### 1. Idle 待机状态
```
┌─────────────────┐
│                 │
│   [精灵图动画]   │  ← 4×4 精灵图，16帧循环播放
│   idle 循环     │
│                 │
└─────────────────┘
```
- 窗口：Normal 模式 (180×180)
- 精灵图：ildesanimation.png
- 帧率：100ms/帧
- 循环：true

### 2. EnterInput 输入状态
```
┌─────────────────┐
│                 │
│   [精灵图动画]   │  ← 3×1 精灵图，3帧循环
│                 │
├─────────────────┤
│ [输入框...]  🎤  │  ← 像素风格输入框
└─────────────────┘
```
- 窗口：Normal 模式
- 精灵图：talkmode.png，帧率 667ms/帧
- 输入框：底部居中，像素边框风格
- 触发：双击 idle/working 状态

### 3. Working 工作状态
```
┌─────────────────────────────┐
│                             │
│   [精灵图动画]        ┌────┤
│   working 循环        │工具1│  ← 白色气泡
│                       │工具2│
│                       └────┘
└─────────────────────────────┘
```
- 窗口：Working 模式 (320×242)
- 精灵图：startworking.png 帧14-48循环
- 气泡：左侧显示工具调用信息

### 4. Response 响应状态
```
┌───────────────────────────────────────┐
│                       ┌──────────────┐
│                       │ [标签名]      │
│   [精灵图动画]        │              │
│   idle 循环           │ Markdown 内容  │
│                       │ (支持代码块/  │
│                       │  表格/列表等)  │
│                       └──────────────┘
└───────────────────────────────────────┘
```
- 窗口：Response 模式 (480×650+，可扩展)
- 响应气泡：右下角显示
- 标签格式：`[标签名] 内容`
- Markdown 渲染：h1-h4、表格、代码块、引用等

### 5. Received 接收完成状态
```
┌─────────────────┐
│   [精灵图动画]   │
│   单帧静态       │
├─────────────────┤
│ 📄 文件1  📄 文件2 │  ← 文件图标列表
│      📄 文件3    │
└─────────────────┘
```
- 窗口：Normal 模式
- 精灵图：received.png 单帧
- 文件覆盖层：顶部居中，最多3个

### 6. Edge 边缘隐藏状态
```
隐藏状态（不可见）：

◄────────────────────────────────►
左边缘                              右边缘
  ↓                                 ↓
┌──────┐                      ┌──────┐
│隐藏区│                      │隐藏区│
└──────┘                      └──────┘

窥视状态（显示部分）：

◄──────►                        ◄──────►
左peek                           右peek
  ↓                                 ↓
┌──────┐                      ┌──────┐
│ 猫猫 │                      │ 猫猫 │ (镜像)
└──────┘                      └──────┘
```
- 窗口：Compact 模式 (180×180)
- 隐藏区：仅显示 65% 宽度
- 窥视区：显示 50% 宽度
- 动画：边缘进入/离开过渡动画

### 7. Dragging 拖动状态
```
┌─────────────────┐
│                 │
│   [精灵图动画]   │  ← 轻微亮度/饱和度变化
│   2帧循环       │
│                 │
└─────────────────┘
```
- 窗口行为：dragging 状态
- 样式：brightness(1.04) saturate(0.96)

## 气泡样式对比

| 类型 | 用途 | 样式特点 |
|------|------|----------|
| working-bubble | 工具调用状态 | 白色圆角，顶部右侧 |
| queue-bubble | 错误/队列消息 | 同上，顶部居中 |
| response-bubble | AI 响应内容 | 米黄底色，右侧，Markdown |
