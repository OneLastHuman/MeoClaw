# 窗口配置

## 基础窗口参数

| 属性 | 值 | 说明 |
|------|-----|------|
| 窗口标题 | MeoClaw | 应用标识名 |
| 窗口宽度 | 360px | 基础宽度（macOS 框架尺寸） |
| 窗口高度 | 360px | 基础高度（macOS 框架尺寸） |
| 猫咪显示尺寸 | 180px | CSS 缩放后实际显示的猫大小 |
| 装饰 | false | 无边框、无标题栏 |
| 透明背景 | true | 支持透明窗口 |
| 置顶 | true | 始终在其他窗口之上 |
| 可调整大小 | false | 固定尺寸 |
| 任务栏显示 | false | 不显示在 Dock/任务栏 |
| 阴影 | false | 无窗口阴影 |

**注**：窗口基础尺寸为 360×360，但内部 CSS 使用 0.5 倍缩放使猫咪显示为 180×180。

## 窗口尺寸模式

窗口根据当前状态动态调整大小：

### Normal 模式
- 窗口尺寸：360 × 360（macOS 框架）
- 猫咪显示：180 × 180（CSS scale=0.5）
- 用于：idle、EnterInput、shock、EnterReceiving、Receiving、received、Response

### Working 模式
- 窗口尺寸：动态扩展
- 猫咪显示：180 × 180
- 气泡显示在工作状态区域

### Response 模式
- 窗口尺寸：1120 × 1480（macOS 框架）
- 用于：Response 状态（显示响应气泡）

## 缩放系统

| 参数 | 最小值 | 最大值 | 步进 | 默认值 |
|------|--------|--------|------|--------|
| petScale | 0.5 | 2.0 | 0.1 | 1.0 |

### 选项面板滑块

选项面板（Options）中 SIZE 滑块使用百分比直观显示：

| Slider 值 | 实际 scale | 说明 |
|-----------|-----------|------|
| 50 | 0.5x | 最小缩放 |
| 100 | 1.0x | 默认大小 |
| 150 | 1.5x | 中间档位 |
| 200 | 2.0x | 最大缩放 |

**注意**：Slider 内部使用 50-200 范围（百分比），通过 `sliderToScale()` 转换为 0.5-2.0 的实际缩放值，确保 100% 位置对应 fill 进度条 50% 位置。

缩放通过 localStorage key `meoclaw.petScale` 持久化。

## 边缘停靠行为

窗口支持吸附到屏幕左右边缘：

| 状态 | 描述 | 触发方式 |
|------|------|----------|
| normal | 正常显示，可自由拖动 | 默认状态 |
| dragging | 拖动中 | 拖动窗口时 |
| edgeHiddenLeft | 隐藏到左边缘 | 拖到左边缘阈值内 |
| edgeHiddenRight | 隐藏到右边缘 | 拖到右边缘阈值内 |
| edgePeekLeft | 显示左边缘窥视 | 鼠标进入左隐藏区 |
| edgePeekRight | 显示右边缘窥视 | 鼠标进入右隐藏区 |

### 边缘阈值参数
- 吸附阈值：windowWidth × 1.0
- 隐藏区可显百分比：65%
- 窥视区可显百分比：50%

## 选项窗口参数

选项窗口（Options）为独立的设置面板窗口：

| 属性 | 值 | 说明 |
|------|-----|------|
| 窗口标题 | MeoClaw 选项 | 用于 Mission Control / 辅助功能标识 |
| 窗口尺寸 | 680 × 550 | 固定尺寸 |
| 最小尺寸 | 620 × 480 | 缩放下限 |
| 装饰 | false | 无边框、无系统标题栏 |
| 透明背景 | true | 支持自定义圆角，透过桌面背景 |
| 置顶 | true | 始终在其他窗口之上 |
| 任务栏显示 | false | 不显示在 Dock/任务栏 |
| 可调整大小 | false | 固定尺寸 |
| 可最大化 | false | 禁止最大化 |
| 可最小化 | false | 禁止最小化 |

**无边框实现**：
- Rust 端：`.decorations(false)` + `.transparent(true)`
- 前端通过 `data-tauri-drag-region` 在 title bar 区域支持窗口拖拽
- 左侧红色圆形关闭按钮调用 `getCurrentWindow().close()`
- 内容区使用 CSS `border-radius: 12px` 实现圆角，通过窗口透明背景透视生效

## CSS 变量系统

通过 `--pet-scale` 控制所有元素的相对大小：

```
--window-width, --window-height    窗口尺寸
--cat-width, --cat-height          猫咪精灵图尺寸（180）
--input-bottom, --input-width       输入框位置
--input-height, --input-font-size   输入框高度和字体
--input-step                        像素边框步长
--file-drop-top, --file-drop-gap    文件拖放提示位置
--file-icon-size, --file-name-font-size  文件图标和名称样式
--close-button-size, --close-button-offset  关闭按钮样式
--queue-bubble-top                  队列气泡顶部位置
--pet-scale                         当前缩放比例
--working-bubble-*                  工作气泡样式
--response-bubble-*                 响应气泡样式
```
