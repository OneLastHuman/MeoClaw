# 组件结构

## 组件层次

```
App
└── DesktopPet.vue           # 主宠物组件（根容器，内嵌文件拖放功能）
    ├── .pixel-bar           # 像素风格输入框
    │   ├── input.pixel-input
    │   └── .mic-btn
    ├── .bubble-overlay      # 工作气泡（工具调用显示）
    │   └── span.bubble-text
    ├── .queue-bubble-overlay # 队列气泡（错误提示）
    │   └── span.bubble-text
    ├── .response-bubble-container  # 响应气泡（Markdown 渲染）
    │   └── .response-bubble
    │       └── .response-bubble-text
    │           ├── span.bubble-envelope
    │           └── div.bubble-markdown
    ├── .desktop-pet          # 精灵图动画元素
    └── canvas.edge-enter-canvas  # 边缘进入动画画布
```

**注**：文件拖放功能内嵌于 DesktopPet.vue 中，无独立的 FileDropOverlay.vue 组件。

## DesktopPet.vue 核心职责

### 状态管理
- `animState: AnimState` - 当前动画状态
- `windowBehaviorState: WindowBehaviorState` - 窗口边缘行为
- `petScale: number` - 缩放比例
- `currentFrame: number` - 当前帧索引

## 文件拖放功能

内嵌于 DesktopPet.vue，处理文件拖放状态的显示：
- 最多显示 3 个文件图标
- 支持文件移除交互
- 位于窗口顶部居中

## 事件监听

| 事件名 | 来源 | 处理函数 |
|--------|------|----------|
| switch-animation | Tauri 后端 | switchAnim(state) |
| update-bubble | Tauri 后端 | 更新工具调用气泡 |
| update-response-bubble | Tauri 后端 | 更新响应内容 |
| pet-scale-changed | Tauri 后端 | 更新缩放 |
| response-sound-changed | Tauri 后端 | 更新音效 |
| drag-drop (onDragDropEvent) | Tauri 窗口 | 处理文件拖放 |

## 样式特点

### 像素风格输入框
使用 CSS box-shadow 模拟 8-bit 像素边框效果：
- 多层 inset shadow 构造像素边缘
- `--input-step: 3px` 控制像素单元
- 支持 hover/active 状态变化

### 响应气泡
- Markdown 渲染（使用 marked 库）
- 支持信封标签 `[标签名]` 解析显示
- 自适应高度（最大 500px）
- 圆角、阴影、边框样式

### 工作气泡（工具调用）
- 单行显示工具调用信息
- 文字过长时显示省略号（`text-overflow: ellipsis`）
- 宽度跟随 `var(--working-bubble-width)`，支持动态缩放

### 精灵图渲染
- CSS background-position 实现帧切换
- `image-rendering: pixelated` 保持像素风格
- 支持缩放（scale transform）
- 隐藏状态支持水平翻转（镜像）

## 响应音效系统

### 音效列表

| ID | 标签 | 文件路径 |
|----|------|----------|
| `off` | Off | - |
| `chime.mp3` | Chime | `/sound/response/chime.mp3` |
| `soft.mp3` | Soft | `/sound/response/soft.mp3` |
| `notification.ogg` | Notification | `/sound/response/notification.ogg` |
| `classic.wav` | Classic | `/sound/response/classic.wav` |
| `crisp.wav` | Crisp | `/sound/response/crisp.wav` |

### 配置存储

- **Storage Key**: `meoclaw.responseSound`
- **默认值**: `chime.mp3`
- **持久化**: localStorage

### 相关函数

```typescript
// 加载音效配置
loadResponseSound(): ResponseSoundId

// 播放音效
getResponseSoundFile(soundId: ResponseSoundId): string | null
```
