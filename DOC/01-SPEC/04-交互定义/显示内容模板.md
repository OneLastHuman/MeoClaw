# 显示内容模板

桌宠在不同动画状态下显示不同的内容，包括输入框、文件图标列表、对话气泡等。

## 状态与显示内容对照表

| 动画状态 | 输入框 | 文件图标 | 对话气泡 | 响应气泡 | 窗口尺寸模式 |
|----------|--------|----------|----------|----------|--------------|
| `idle` | - | - | - | - | `normal` |
| `EnterInput` | ✓ | - | - | - | `normal` |
| `shock` | - | - | - | - | `normal` |
| `startworking` | - | - | - | - | `normal` |
| `working` | - | - | ✓ | - | `working` |
| `workingPreview` | - | - | ✓ | - | `working` |
| `EnterReceiving` | - | - | - | - | `normal` |
| `Receiving` | - | - | - | - | `normal` |
| `received` | ✓ | ✓ | - | - | `normal` |
| `Response` | - | - | - | ✓ | `response` |

## 输入框

### 显示条件

```typescript
const showInput = computed(
  () =>
    !isCompactWindow.value &&
    (animState.value === 'EnterInput' ||
      animState.value === 'received' ||
      animState.value === 'Response'),
);
```

输入框在以下状态显示：
- `EnterInput`: 用户双击后进入输入模式
- `received`: 文件拖拽接收完成后进入
- `Response`: 收到响应时显示（预留）

### 外观

使用像素风格边框的输入框，配合麦克风按钮：

```
┌──────────────────────────────┐
│ ask me anything...  [麦克风]  │
└──────────────────────────────┘
```

- 像素边框风格（4层边框：黑、灰浅、灰深、白）
- 底部定位在窗口下方
- 按 `Enter` 提交内容

### 提交逻辑

```typescript
async function onEnterPress() {
  if (!inputText.value.trim()) {
    cancelFromEnterInput = true;
    switchAnim('working');
    return;
  }

  const text = inputText.value;
  let fullMessage = text;
  if (droppedFiles.value.length > 0) {
    // 附加文件路径到消息
    const attachLines = ['\n\n[附件]'];
    for (const file of droppedFiles.value) {
      attachLines.push(`文件: ${file.path}`);
      attachLines.push(`类型: ${getMimeType(ext)}`);
    }
    fullMessage += attachLines.join('\n');
  }

  await invoke('send_message', { message: fullMessage, attachments: [] });
  inputText.value = '';
  droppedFiles.value = [];
  switchAnim('working');
}
```

### 失焦处理

输入框失焦（点击外部）时，若处于 `EnterInput` 状态则切换回 `idle`：

```typescript
function onInputBlur() {
  if (animState.value !== 'EnterInput') {
    return;
  }
  cancelFromEnterInput = true;
  switchAnim('idle');
}
```

## 文件图标列表

### 显示条件

`animState === 'received' && !isCompactWindow`

### 外观

- 固定显示在窗口顶部上方（`fileDropTop: 70px`）
- 最多显示3个文件图标
- 每个图标下方显示截断后的文件名

```
   [icon1]  [icon2]  [icon3]
   file1    file2    file3
```

每个文件图标右上角有关闭按钮（×），点击移除该文件。

## 对话气泡（Working Bubble）

### 显示条件

```typescript
const showBubble = computed(
  () =>
    !isCompactWindow.value &&
    (animState.value === 'working' || animState.value === 'workingPreview') &&
    bubbleTools.value.length > 0,
);
```

### 内容

显示 OpenClaw 工具调用信息：

```typescript
const bubbleDisplayText = computed(() => {
  if (bubbleTools.value.length === 0) return '';
  return bubbleTools.value
    .map((tool) => (tool.detail ? `${tool.name}: ${tool.detail}` : tool.name))
    .join(' | ');
});
```

### 外观

白色圆角气泡，定位在猫身右上方：

```
  ┌─────────────────────────┐
  │ tool_call: preview mode │
  └─────────────────────────┘
            🐱
```

## 响应气泡（Response Bubble）

### 显示条件

```typescript
const showResponseBubble = computed(() => !isCompactWindow.value && animState.value === 'Response');
```

### 内容解析

响应文本支持 envelope 标记（`[标记名]`）和 Markdown 格式：

```typescript
const ENVELOPE_REGEX = /^\[([^\]]+)\]\s*/;

function parseBubbleContent(text: string): BubbleContent {
  const match = text.match(ENVELOPE_REGEX);
  let envelope: string | null = null;
  let content = text;

  if (match) {
    envelope = match[1];
    content = text.slice(match[0].length);
  }

  return {
    envelope,
    html: marked.parse(content, { breaks: true, gfm: true }) as string,
  };
}
```

### 外观

- 浅米色背景（`#f7f2ec`）
- 左下角定位
- 固定最大高度（`500px`），超出可滚动
- 支持 Markdown 渲染

```
🐱
  ┌──────────────────────────┐
  │ [思考]                  │
  │ Markdown 格式的响应内容  │
  │ - 列表项                 │
  │ - **粗体**               │
  └──────────────────────────┘
```

### 音效

`Response` 状态首次显示时播放提示音（可配置）：

```typescript
function playResponseSound() {
  const soundId = responseSound.value;
  if (soundId === RESPONSE_SOUND_OFF) return;
  const soundFile = getResponseSoundFile(soundId);
  if (!soundFile) return;
  try {
    const audio = new Audio(soundFile);
    audio.play().catch(...);
  } catch (...) {}
}
```

## 窗口尺寸模式

```typescript
export type WindowMode = 'normal' | 'working' | 'response';

export function windowModeForState(state: AnimState): WindowMode {
  if (state === 'Response') return 'response';
  if (state === 'working' || state === 'workingPreview') return 'working';
  return 'normal';
}
```

| 模式 | 窗口宽度 | 窗口高度 |
|------|----------|----------|
| `normal` | `catWidth * scale` | `catHeight * scale` |
| `working` | `workingBubbleLeft + workingBubbleWidth` | `catHeight + workingBubbleTop + workingBubbleMinHeight` |
| `response` | `responseBubbleLeft + responseBubbleWidth` | `responseBubbleBottom + responseBubbleHeight` |

窗口尺寸会随 `petScale` 缩放因子动态调整。`Response` 模式下窗口高度会根据气泡实际内容高度动态调整。

## 边缘行为占位符

当窗口处于贴边隐藏或 peek 状态时，显示专用的占位符精灵图（2帧循环），覆盖在主精灵图上方：

- `dragging`: 拖拽中
- `edgeHiddenLeft/Right`: 贴边隐藏
- `edgePeekLeft/Right`: peek 窥视

占位符使用 canvas 渲染，主精灵图（`desktop-pet` div）透明度设为0。
