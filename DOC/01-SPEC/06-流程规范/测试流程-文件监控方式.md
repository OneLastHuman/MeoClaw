# 测试流程：文件监控方式

## 概述

通过文件写入触发状态切换，Rust 后端监控文件变化并发射 Tauri 事件，前端接收事件后切换动画。

## 架构

```
终端命令                      Rust 后端                   前端
    │                           │                          │
    ▼                           ▼                          ▼
echo "状态" > /tmp/      →   文件监控线程     →   Tauri 事件   →   switchAnim()
meo_anim_state.txt           读取文件内容        发射事件        更新 UI
```

---

## 状态切换命令

```bash
echo "idle" > /tmp/meo_anim_state.txt
echo "EnterReceiving" > /tmp/meo_anim_state.txt
echo "Receiving" > /tmp/meo_anim_state.txt
echo "startworking" > /tmp/meo_anim_state.txt
echo "working" > /tmp/meo_anim_state.txt
echo "EnterInput" > /tmp/meo_anim_state.txt
```

## 校验清单

| 步骤 | 命令 | 预期结果 |
|------|------|----------|
| 1 | `echo "EnterReceiving" > /tmp/meo_anim_state.txt` | 单次播放，停在最后一帧 |
| 2 | `echo "Receiving" > /tmp/meo_anim_state.txt` | 循环播放 |
| 3 | `echo "startworking" > /tmp/meo_anim_state.txt` | 播放完自动切换到 working |
| 4 | `echo "idle" > /tmp/meo_anim_state.txt` | 回到 idle 状态 |

## 关键文件

### Rust 后端
- `src-tauri/src/lib.rs` - 文件监控 + 事件发射
- `src-tauri/capabilities/default.json` - 需要 `core:event:default` 权限

### 前端
- `src/components/DesktopPet.vue` - 监听 `switch-animation` 事件
- `src/config/animations.ts` - 动画配置，包含 `nextState` 字段

### 测试
- `src/config/__tests__/animations.test.ts` - Vitest 单元测试
- `vitest.config.ts` - Vitest 配置

## 技术要点

### 1. Rust 文件监控

使用 `notify` crate 监控文件变化：

```rust
use notify::{Watcher, RecursiveMode};
use std::sync::mpsc;
use std::time::Duration;

Config::default().with_poll_interval(Duration::from_millis(500))
```

### 2. Tauri 事件发射

```rust
app.emit("switch-animation", state)
```

### 3. 前端事件监听

```typescript
import { listen } from '@tauri-apps/api/event';

unlisten = await listen<string>('switch-animation', (event) => {
    const state = event.payload as AnimState;
    if (state && ANIMATIONS[state]) {
        switchAnim(state);
    }
});
```

### 4. 动画配置中的 nextState

```typescript
startworking: {
    loop: false,
    nextState: 'working',  // 播放完自动切换
},
EnterReceiving: {
    loop: false,
    // 无 nextState，播放完停止
},
```

---

## 注意事项

1. **Tauri 2 事件权限** - 必须在 `capabilities/default.json` 添加 `core:event:default`
2. **前端热更新** - 修改 Vue 文件后自动生效，无需重启
3. **Rust 代码修改** - 需要重启进程重新编译
